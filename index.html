Move Ranking Project

This is the first project of many from Angela Yu's udemy course on web development. The point of the project was to use the Header Tags to display important information correctly, as well as to utilize the Paragraph Tag for easy to read text.
As a bonus we also learned about Void Elements, which are self closing Tags fulfilling a specific purpose. For example, the hr tag creates a horizontal line.

Birthday Invite Project

Next up is the Birthday Invite Project. The goal of this project is to utilize lists, unsorted lists in this case, and use the Anchor Tags and Image Elements to create a personal birthday invite. The knowledge of the previous project is also required.
The image of the birthday cake is an external link, which has to be included using the src attribute. The Google Maps link is the Anchor Tag and leads to a surprising gathering of the minds. Exciting!
